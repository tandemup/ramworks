Política de privacidad de Shopp

Última actualización: 27 de mayo de 2026

Shopp es una aplicación Android para crear listas de compra, organizar productos, escanear códigos de barras y consultar información de productos mediante motores de búsqueda externos.

1. Datos que puede guardar la app

La app puede guardar en el dispositivo información introducida por el usuario, como listas de compra, nombres de productos, categorías, tiendas favoritas, preferencias de búsqueda e historial de escaneos si el usuario decide utilizar esa función.

2. Almacenamiento local

La información se almacena principalmente de forma local en el dispositivo. Shopp no requiere crear una cuenta de usuario para utilizar sus funciones básicas.

3. Cámara y escaneo de códigos

La app puede solicitar acceso a la cámara para escanear códigos de barras. El acceso a la cámara se utiliza únicamente para realizar el escaneo.

4. Motores de búsqueda externos

Cuando el usuario decide buscar información sobre un producto, la app puede abrir servicios externos como buscadores web o bases de datos de productos. Esos servicios pueden aplicar sus propias políticas de privacidad.

5. Compartición de datos

Shopp no vende datos personales del usuario. Si la app abre servicios externos, la interacción con esos servicios queda sujeta a sus propias condiciones y políticas.

6. Eliminación de datos

El usuario puede eliminar los datos guardados localmente desde las opciones de la app, si dichas opciones están disponibles, o desinstalando la aplicación del dispositivo.

7. Contacto

Para consultas sobre privacidad o soporte, puedes escribir a: tu-email@dominio.com
